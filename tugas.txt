Merangkum semua materi Git buat dalam bentuk commit

Pertemuan Kedua : Fundamental

    - Git Init
        Digunakan untuk menginisialisasi repository atau memberi tahu folder saat ini akan dijadikan repository kita tidak bisa menggunakan git tanpa menginisialisasinya

    - Git Config
        Konfigurasi email dan username biasanya disamakan dengan akun gitlab / github

    - Git status
        Digunakan untuk mengetahui status file
    
    - Git clone
        Digunakan untuk mengcopy repo dari server

    - Git push
        Untuk mengirimkan commit ke server

    - Git remote
        Semacam api untuk berinteraksi dari client dengan server
    
    - Git pull
        Untuk mengambil commit dari server jika ada perubahan di server yang dibuat oleh branch lain
